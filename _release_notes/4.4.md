<b>\*\*\* Squeak 4.4 &quot;Ulam Spiral&quot; Release Notes \*\*\*</b>Released 2012/12/31: <a href="http://ftp.squeak.org/4.4/Squeak4.4-12327.zip">http://ftp.squeak.org/4.4/Squeak4.4-12327.zip</a>

- <b>Cleanup</b> and simplification of <b>Morphic text editing</b>
- <b>Add host window support</b>, letting you display things in a host window. (Currently only on Windows and Mac.)
- <b>Bugfixes</b> in the <b>Compiler</b>, <b>Parser</b> and <b>Debugger</b> toolchain
- Stub support for the ability to <b>evolve</b> the <b>bytecode set</b>
- <b>Better printing</b> of <b>Floats</b>, hashing of <b>DateAndTimesChangeSorter</b> improvements
- <b>ToolBuilder</b> improvements
- Merged <b>network improvements</b> from Etoys
- <b>Monticello browsing</b> can now group versions <b>by branch</b>, making tracking of parallel developments easier
- <b>Decreased coupling</b> between core packages<b>I</b>
- <b>Pv6 support</b> (if your VM provides it)
