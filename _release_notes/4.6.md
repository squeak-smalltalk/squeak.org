<b>\*\*\* Squeak 4.6/5.0 Release Notes \*\*\*</b>
This is a dual release because of the introduction of the new Spur object format. Squeak 4.6 images have the former <b>V3</b> format while Squeak 5.0 images have the <b>Spur</b> format.
